# Python Date & Time Setter GUI

A Python Tkinter GUI application to view, check, and set system date and time. 
Includes real-time system clock display, error handling for invalid input, and easy-to-use graphical interface.

## Features

- View current system date and time
- Check if system date/time matches user input
- Set system date/time if incorrect
- Real-time system clock display
- User-friendly GUI with message pop-ups
- Error handling for invalid inputs
